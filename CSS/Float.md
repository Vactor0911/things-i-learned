# Float
`float` 속성이 적용된 HTML 요소는 원래의 흐름에서 벗어나 둥둥 떠다니듯 배치됨  
인접한 텍스트 또는 인라인 요소가 그 주위를 자연스럽게 감쌈  
[더 알아보기](https://developer.mozilla.org/ko/docs/Web/CSS/float)

**자주 사용하는 속성값**
- `none` : 기본값으로 요소를 띄우지 않음
- `left` : 왼쪽에 띄움
- `right` : 오른쪽에 띄움
- `inherit` : 부모 요소로부터 상속
- `initial` : 기본값으로 설정

<br />

# Clear
`float: left;` 혹은 `float: right;` 값을 취소함
- `none` : 기본값
- `left` : 왼쪽을 취소
- `right` : 오른쪽을 취소
- `both` : 양쪽을 모두 취소
